## 🧭 What this app does

Clean-Social-Link is a very lightweight Android app that helps you clean shared social-media URLs.

When you copy a link from a social app, it often includes extra tracking parts. This app removes those extra parts so you can share a cleaner link.

Use it when you want to:
- remove long tracking strings from shared links
- keep links short and tidy
- copy a cleaned link with less clutter
- avoid sending extra data in a URL
